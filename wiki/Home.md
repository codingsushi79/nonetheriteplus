# NoNetheritePlus Wiki

Welcome to the **NoNetheritePlus** plugin documentation! This wiki contains comprehensive information about installing, configuring, and using the NoNetheritePlus plugin for your Minecraft server.

## 📦 About NoNetheritePlus

NoNetheritePlus is a powerful Minecraft plugin that provides configurable netherite crafting restrictions with advanced features like material refunds, statistics tracking, and world-specific configurations.

### ✨ Key Features
- **Configurable Crafting Restrictions** - Control which netherite items can be crafted
- **Material Refund System** - Automatically refund materials when crafting is blocked
- **Template Control** - Prevent netherite upgrade templates from spawning
- **Statistics & Analytics** - Track plugin usage and player behavior
- **World-Specific Settings** - Different configurations per world
- **Debug Mode** - Enhanced logging for troubleshooting
- **Configuration Presets** - Save and load different server profiles
